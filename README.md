# Team Task Manager

A full-stack Team Task Management Web Application built using Node.js, Express.js, Firebase Firestore, JWT Authentication, Railway, and Vercel.

## Features

- User Authentication (Signup/Login)
- JWT-based secure login
- Role-Based Access Control (Admin / Member)
- Project Management
- Task Creation & Assignment
- Task Status Tracking
- Dashboard Analytics
- Overdue Task Tracking
- Manage Members
- Responsive UI

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- Vercel Deployment

### Backend
- Node.js
- Express.js
- JWT Authentication
- Railway Deployment

### Database
- Firebase Firestore (NoSQL)
